# Anima Prompt Studio

Category: `EasyUse Anima/Prompt`

This node has the same outputs and prompt-building behavior as `Anima Prompt
Builder`, but adds editing helpers inside the node UI.

![Anima Prompt Studio](../images/nodes/anima-prompt-studio.png)

## Main Features

- Places the LoRA trigger input near the top of the node.
- Lets prompt text fields be resized vertically in the node UI.
- Applies tag autocomplete and category highlighting to text fields.
- Reflects externally connected text in the preview and highlight overlay after
  workflow execution.
- Keeps edited text in workflow serialization.
