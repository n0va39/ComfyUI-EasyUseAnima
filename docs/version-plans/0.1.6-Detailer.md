# EasyUseAnima 0.1.6 Detailer / SAM3 계획

## 현재 결정

0.1.6에서는 Impact Pack의 detailer core를 EasyUse Anima 내부에 복제하지 않는다.

EasyUse Anima가 담당하는 범위는 다음으로 제한한다.

- Impact Pack `DETAILER_HOOK` 호환 hook 제공
- ANIMA/Spectrum/특수 VAE 워크플로우에서 crop sampling 크기를 32배수 등으로 정렬
- ComfyUI native SAM3와 Impact Pack `MaskToSEGS` / `DetailerForEach` 연결을 편하게 쓰는 편의 노드 제공
- Impact Pack 미설치 상태에서도 EasyUse Anima의 다른 노드는 import 가능하게 유지

## 제거한 이전 계획

다음 계획은 폐기한다.

- Impact Pack `DetailerForEach` core 재구현
- Impact Pack `MaskToSEGS` core 재구현
- EasyUse Anima 내부 SEGS detail loop 구현
- sampling backend, tiled encode/decode, wildcard cycle, scheduler function 처리 직접 재현
- Impact Pack을 대체하는 독립 detailer 구현

이전 계획은 유지보수 범위가 너무 크고, Impact Pack 업데이트와 충돌할 가능성이 높다. 실제 문제는 crop sampling 크기가 latent 경로에서 맞지 않는 것이므로 hook으로 해결한다.

## 공개 노드

### Anima Detailer Align Hook

- class: `EasyUseAnimaDetailerAlignHook`
- display: `Anima Detailer Align Hook`
- output: `DETAILER_HOOK`

역할:

- Impact Pack `detailer_hook.touch_scaled_size(width, height)` 경로를 감싼다.
- 기존 `detailer_hook`이 있으면 기존 hook을 먼저 실행한 뒤 크기 정렬만 추가한다.
- 기본 `alignment=32`를 권장한다.
- `alignment=none`이면 원본 크기를 유지한다.

검증된 문제:

- 예: `1052 x 1232` crop sampling 크기는 32배수가 아니어서 latent size mismatch를 만들 수 있다.
- hook 적용 시 `1056 x 1248`처럼 정렬되어 16채널 VAE/Spectrum 경로에서 오류를 줄인다.

### Anima SAM3 Context

- class: `EasyUseAnimaSAM3Context`
- display: `Anima SAM3 Context`
- output: `ctx_SAM3`, `MODEL`, `CLIP`, `VAE`

역할:

- SAM3 모델/CLIP/VAE context를 하나로 묶어 SAM3 편의 노드에서 사용한다.
- 다른 노드에 MODEL/CLIP/VAE를 그대로 전달할 수 있게 한다.

### Anima SAM3 Detailer

- class: `EasyUseAnimaSAM3Detailer`
- display: `Anima SAM3 Detailer`

역할:

- ComfyUI native SAM3 detection을 사용한다.
- mask to segs 변환은 Impact Pack `MaskToSEGS`에 위임한다.
- detail loop는 Impact Pack `DetailerForEach`에 위임한다.
- EasyUse Anima는 연결 편의성과 오류 메시지 정리만 담당한다.

## 의존성 정책

- Impact Pack은 Python package dependency가 아니라 ComfyUI custom node 런타임 의존성이다.
- `pyproject.toml` dependencies에는 넣지 않는다.
- Impact Pack이 없어도 EasyUse Anima import는 실패하지 않아야 한다.
- Impact Pack이 필요한 SAM3 detailer 실행 시점에만 명확한 오류를 낸다.

오류 메시지는 다음을 포함해야 한다.

- 필요한 커스텀 노드: `ComfyUI-Impact-Pack`
- 필요한 노드: `MaskToSEGS`, `DetailerForEach`
- 대체 방법: Impact Pack 설치 또는 해당 기능을 쓰지 않는 워크플로우 사용
