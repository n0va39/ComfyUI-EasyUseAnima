## First Run

1. Install the required models and custom nodes listed in the guide.
2. Load the workflow file in ComfyUI.
3. Run the base generation path first with optional optimization blocks disabled if your environment is unstable.
4. Enable HighRes, Inpainting, Detailer, and Upscale sections only after the base path works.

The workflow itself includes section guide notes at the top of each major group.
